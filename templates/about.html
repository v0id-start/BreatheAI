About Convolutional Neural Networks & Machine Learning



In machine learning, a neural network is a collection of neurons connected to loosely model a human's brain. Neural networks are able to 'learn' by adjusting weights and biases between layers of neurons during training to alter the output.

Source: https://towardsdatascience.com/a-comprehensive-guide-to-convolutional-neural-networks-the-eli5-way-3bd2b1164a53


The input image is first flattened converted into a multi-dimensional array of vectors that represent the color value of each pixel in RGB. Convolutions are performed on the input to extract high level details of the image that it can use to distinguish possible outputs. After many convolutions, the output is flattened to provide an output. With convolutional neural networks, this is usually an identification of an object within an image.
